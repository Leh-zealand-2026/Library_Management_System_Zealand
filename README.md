# Library_Management_System_Zealand

The program requires python.

type "python --version" to see if you have it installed, otherwise you can install it from the official python website.

To run main program:

1. Type python main.py in terminal to run main file.
2. Follow the menu instructions that shows up in terminal to use the system.
3. For test cases use pytest, type "pip install pytest" into terminal to install pytest.
4. Write pytest test_library.py to run test cases.


This project is for a programming assignment, description of required functions below.

Obligatorisk Opgave No 1

From the assignment description:

Objective: 
Develop a library management system using Python that demonstrates your understanding of OOP concepts such as classes, objects, inheritance, and polymorphism.

Requirements:

Classes:Book class, Member class, Library class

Attributes: books, members

Methods: add_book, remove_book, update_book, add_member, 
remove_member, update_member, issue_book, return_book, 
display_books, display_members 

Functions:
Book Management: 
● Add a new book to the library. 
● Remove a book from the library by its ID. 
● Update the details of an existing book (title, author, copies). 
● Display all books in the library

● Member Management: 
● Add a new member to the library. 
● Remove a member from the library by their ID. 
● Update the details of an existing member (name). 
● Display all members of the library

● Book Issuing and Returning: 
● Issue a book to a member if copies are available. 
● Return a book to a member.

● Display Information: 
● Display detailed information about books and members.
